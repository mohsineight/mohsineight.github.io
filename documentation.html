Introduction

Shades is a simple 5-color palette generator. This blog acts as a guide for anyone who is looking to deploy a simple machine learning/ deep learning app on web. Though there are countless guides and blogs available for deploying a machine learning app on web, the purpose of this guide is to provide with a simple and robust solution for a lightweight machine learning model deployment.

About this App

The deep learning model is developed and trained using Keras (Tensorflow backend) and is deployed using simple HTML/CSS/Javascript based web interface. The web app is deployed on github pages.


Getting Started


Prerequisites

Before starting it is assumed that you are familiar with:

    (For model training)

  • python, Keras and jupyter notebook.
  • deep learning theory, specifically GANs.

    • (For model deployment)

  • basic HTML, CSS and Javascript.

Important: This is not a guide on deep learning training methodologies or front-end development. The approach used for this app is/can-be unoptimized but the main purpose to provide with a reference point; combining basic data processing, model development, HTML/CSS/Javascript development for simple AI based web app deployment. This guide is for educational purposes only.


Setting up environment

The process described here can be followed using any Operating System. The small details (such as setting up environment, installing necessary packages etc.) are purposefully left as reader can find detailed guide on internet. To start working on the model training, jupyter notebook is used. To start working on a jupyter notebook it is assumed that:

  1. You have a separate virtual environment with python, numpy, pandas, matplotlib, opencv, tensorflow and jupyter notebook installed.

  2. You have dataset downloaded (if following this app) or have your own dataset processed (if working on different app).
